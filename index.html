Track products, manage locations, and control stock across your entire supply chain. From warehouse to vessel, LibreStock keeps everything in order.

Get Started See Features 
$ git clone https://github.com/librestock/librestock.git
$ cd librestock
$ pnpm install
$ pnpm dev

  LibreStock API    → http://localhost:3000/api/v1
  LibreStock Web    → http://localhost:5173
  Swagger Docs      → http://localhost:3000/api/docs

Everything you need to manage provisions

A complete toolkit for tracking inventory across warehouses, suppliers, and vessels.

Product Management

Create and organize products with SKUs, pricing, and multi-level category hierarchies with unlimited nesting.

Location Tracking

Manage warehouses, suppliers, clients, and in-transit locations. Define zones, shelves, and bins within each.

Inventory Control

Track stock quantities across locations with batch tracking, expiry dates, and real-time availability.

Order Processing

Track yacht provisioning orders through their complete lifecycle from request to delivery.

Audit Trail

Complete change history with user tracking and soft delete support. Know who changed what and when.

Auth & RBAC

Built-in authentication with role-based access control. Manage users, roles, and permissions out of the box.

Multi-Language

Internationalized from the ground up with English, French, and German translations. Easy to extend.

REST API with Docs

Full HATEOAS-enabled REST API with Swagger/OpenAPI documentation, rate limiting, and pagination.

Up and running in minutes

Clone, install, and start developing. It's that simple.

1

Clone the repo

git clone https://github.com/librestock/librestock.git
cd librestock
2

Install dependencies

pnpm install
3

Configure environment

cp backend/.env.template backend/.env
cp frontend/.env.example frontend/.env
4

Start developing

pnpm dev
